# Crud Application Using 
* Node
* Typescript
* Mongo
* EJS
* Test Cases Using Chai

> Node app where we can perform
* GET
* POST
* PUT
* DELETE

Operations data is saved in mongoDB.

Step To Run App
> Install Nodejs
> Install MongoDb
> run mongod
> npm run dev
> http://localhost:4000/users

To Run test Case
> npm run test
